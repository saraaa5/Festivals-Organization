# Festival Organizer Platform

A responsive web application that centralizes festival organizers and their events in one place.  
Users can browse festival organizers, explore festivals, and administrators can manage platform content through dedicated dashboards.

## Features

### Public Pages
- Homepage with a list of festival organizers
- Organizer details page
- Festival details page
- Login popup
- Registration popup
- Custom error pages

### Admin Dashboard
- View all festival organizers
- Edit organizer information
- Delete organizers
- Add/edit/delete festivals for each organizer
- View all registered users
- Edit user information
- Delete users

## Search Functionality

### Organizer Search
Users can search festival organizers by name directly from the homepage.

### Festival Search
Users can search festivals by:

- name
- type
- location

Search filters can be used individually or combined.

### Keyword Highlighting
Matched search terms are highlighted inside search results for better user experience.

## Form Features

- Client-side validation
- Pre-filled edit forms
- Confirmation dialogs for delete actions
- Registration form
- Login form

## Responsive Design

The application is fully responsive and optimized for:

- desktop devices
- tablets
- mobile devices

## Data Management

Dynamic data is fetched from Firebase and used for:

- festival organizers
- festivals
- users
